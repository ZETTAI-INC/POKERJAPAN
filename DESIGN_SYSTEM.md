全ページで統一するUI/UX仕様。新規ページ作成時は必ずこのドキュメントに従うこと。

## 3. 共通レイアウト

### コンテナ
```css
.pj-width { max-width:1200px; margin:0 auto; padding:0 24px; width:100%; }
```

### セクション
```css
.pj-section { padding:64px 0; }          /* 白背景 */
.pj-section-alt { background:#f7f8fa; }  /* グレー背景（交互使用） */
```

### セクションヘッダー
```css
.pj-section-header { display:flex; justify-content:space-between; align-items:center; margin-bottom:32px; }
```
左にタイトル、右に「もっと見る →」リンク。

## 4. ヘッダー（全ページ共通）

- 高さ: 64px
- 背景: `rgba(18,18,20,.97)` + `backdrop-filter:blur(12px)`
- sticky配置
- 構成: ロゴ（左）| ナビ（中央）| ボタン群（右）| ハンバーガー（モバイル）
- ロゴ: `filter:brightness(0) invert(1)` で白表示
- ナビリンク: `#adb1b8` → ホバー `#fff` / アクティブ `#fff`
- ボタン: ログイン（ghost）+ 無料登録（gold背景）

### ナビ項目（5つ）
| ラベル | リンク先 |
|---|---|
| ニュース | `news.html` |
| 海外カジノ | `casinos.html` |
| 戦略ガイド | `strategy.html` |
| トーナメント | `tournaments.html` |
| 初心者向け | `beginners.html` |

### アクティブ状態
現在のページに該当するナビリンクに `class="active"` + `color:#fff` を適用。

## 5. モバイルメニュー

- 768px以下でナビ・ボタン非表示、ハンバーガーアイコン表示
- フルスクリーンオーバーレイ: `rgba(18,18,20,.98)`
- メニューリンク: 16px, `#adb1b8`, 下ボーダー区切り
- 下部にログイン・無料登録ボタン横並び
- ハンバーガーアイコン: 3本線 → X トグルアニメーション

---

## 6. サブページヒーロー

ホームのダークヒーローの簡易版。各タブページ共通で使用。

```css
.page-hero {
  background:#0b0e11; color:#fff;
  padding:48px 0; position:relative; overflow:hidden;
}
```
- パンくずリスト: `ホーム > [ページ名]`（13px, `#595d61`）
- タイトル: h1, 40px, 700
- サブタイトル: 16px, `#adb1b8`
- 右上に薄い放射グラデーション装飾

## 7. コンポーネント一覧

### 7-1. ニュースカード `.pj-news-card`
- 白背景、角丸16px、薄ボーダー
- サムネイル: `aspect-ratio:16/9`、`object-fit:cover`
- ボディ: padding 20px 24px
- タグ: 11px uppercase, 色分けバッジ
  - `.tournament` → 背景 `#fef3e2` 文字 `#b8750c`
  - `.cash` → 背景 `#e8f5ee` 文字 `#1a6b3c`
  - `.strategy` → 背景 `#eef0ff` 文字 `#4a5eb8`
  - `.beginner` → 背景 `#ffeaeb` 文字 `#c0392b`
- フィーチャードカード: `grid-column:1/-1`, 2カラム（画像+テキスト）
- グリッド: `grid-template-columns:1fr 1fr; gap:20px`
- ホバー: `box-shadow:0 4px 20px rgba(0,0,0,.06)` + `translateY(-2px)`

### 7-2. プロダクトカード `.pj-product-card`
- 白背景、角丸16px、薄ボーダー
- アイコン画像: 64x64px, 角丸8px, `mix-blend-mode:multiply`
- タイトル: 16px 600
- 説明: 13px `--text-3`
- グリッド: 4カラム（タブレット2、モバイル2、極小1）
- ホバー: box-shadow + border-color変更

### 7-3. テーブル `.pj-table`
- ヘッダー: 12px uppercase, `--text-3`, 下ボーダー
- 行: 14px, hover時 `--bg-alt` 背景
- ステークス列: `--accent` 色, 600ウェイト
- モバイル: `.pj-table-wrap` で横スクロール

### 7-4. CTAバナー `.pj-cta`
- グリーングラデーション背景: `linear-gradient(135deg,#1a6b3c,#0f4025)`
- 角丸20px、padding 56px 48px
- 白テキスト、flex横並び
- ボタン: 白背景、accent色テキスト

### 7-5. タグバッジ `.pj-news-tag`
追加カラー定義:
- `.online` → 背景 `#e8f0ff` 文字 `#2563eb`
- `.report` → 背景 `#fef3e2` 文字 `#b8750c`
- `.gto` → 背景 `#f3e8ff` 文字 `#7c3aed`
- `.basic` → 背景 `#e8f5ee` 文字 `#1a6b3c`
- `.mental` → 背景 `#fff3e0` 文字 `#e65100`

## 10. ページ別定義

### 10-1. ニュース (`news.html`)
- ヒーロー: 「最新ニュース」タイトル + 「ポーカー業界の最新情報をお届けします」サブ
- フィルタータブ: すべて / トーナメント / キャッシュゲーム / 戦略 / 初心者（`.pj-news-tag`スタイルのタブ）
- メインコンテンツ: `.pj-news-grid` で記事一覧（8-12記事）
- 各記事: サムネ + タグ + タイトル + 概要 + 日付
- サイドバーなし（フルワイド2カラムグリッド）
- CTA: メルマガ登録バナー

### 10-2. 海外カジノ (`casinos.html`)
- ヒーロー: 「海外カジノ・ポーカールーム」タイトル + 統計（50+国、1200+ルーム）
- リージョンタブ: すべて / アジア / アメリカ / ヨーロッパ / オセアニア
- カジノカード: 国旗 + カジノ名 + 都市 + ステークス範囲 + ゲーム数 + ステータスバッジ
  - カードレイアウト: `.pj-product-grid` 3カラム
  - 各カードクリック → `junkets/xxx.html` へ
- 下部マップ風エリア（テキストベース世界地図、主要都市にドット）
- CTA: 「お問い合わせ」バナー

### 10-3. 戦略ガイド (`strategy.html`)
- ヒーロー: 「戦略ガイド」タイトル + 「初心者からプロまで、レベル別の戦略記事」サブ
- レベルフィルター: すべて / 初級 / 中級 / 上級
- 記事カード: `.pj-news-card` 再利用
  - タグ色: 初級→green / 中級→gold / 上級→red
- カテゴリセクション:
  - プリフロップ戦略
  - ポストフロップ戦略
  - メンタル・バンクロール
  - GTO理論
- 各カテゴリ `.pj-section` / `.pj-section-alt` 交互
- CTA: 「メルマガで限定戦略記事を受け取る」

### 10-4. トーナメント (`tournaments.html`)
- ヒーロー: 「トーナメント速報」タイトル + 「世界の主要ポーカートーナメントを追う」サブ
- 開催中トーナメント: カード形式（日程 + 場所 + バイイン + 賞金プール）
  - ステータスバッジ: 開催中(green) / 近日開催(gold) / 終了(gray)
- トーナメントカレンダー: テーブル形式（月別）
- 日本人プレイヤー成績: テーブル（名前 / トーナメント / 順位 / 賞金）
- シリーズ別リンク: WSOP / WPT / EPT / APPT（4カラムカード）
- CTA: 「トーナメント情報をいち早く受け取る」

### 10-5. 初心者向け (`beginners.html`)
- ヒーロー: 「はじめてのポーカー」タイトル + 「ゼロからわかる海外ポーカーガイド」サブ
- ステップガイド: 番号付きステップ（1→2→3→4→5）
  1. ポーカーの基本ルール
  2. ハンドランキングを覚える
  3. 基本戦略を学ぶ
  4. 国内で練習する
  5. 海外デビューする
- 各ステップ: アイコン + タイトル + 説明（縦並びタイムライン風）
- よくある質問 (FAQ): アコーディオン形式
- おすすめ初心者向けカジノ: `.pj-product-card` 3枚（Paradise City, Manila COD, Star Sydney）
- CTA: 「無料で始める」

  └── reference/

      ├── fonts/
